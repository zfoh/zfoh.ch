ZuriHac 2026 / Projects

This is the list of projects that attendees of ZuriHac 2026 are planning to work on. If any of these spark your interest, then you will be able to join the project channel on Discord or find the contact at the venue and hack away on it with other participants.

If you want to get your own project included, please add your project name and description by submitting a pull request here.
