Projects

This is the list of projects that attendees of ZuriHac 2020 are planning to work on. If any of these spark your interest, then you will be able to join the project channel on Discord and hack away on it with other participants.

If you want to get your own project included, please add your project name and description by submitting a pull request here. You will get your very own project channel on our Discord server, and other people can join you.
